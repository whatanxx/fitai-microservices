# User Service

> **Coming in Week 3** – User Service – rejestracja, logowanie, JWT auth

Serwis odpowiedzialny za zarządzanie użytkownikami aplikacji FitAI.

## Planowana funkcjonalność

- Rejestracja nowych użytkowników
- Logowanie i wystawianie tokenów JWT
- Zarządzanie profilem użytkownika (dane fitness: waga, wzrost, poziom)

## Stack

- Python 3.11 + FastAPI
- PostgreSQL (SQLAlchemy + Alembic)
- JWT (python-jose)

`8001`

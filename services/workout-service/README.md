# Workout Service

> **Coming in Week 3** – Workout Service – CRUD planów treningowych i ćwiczeń

Serwis odpowiedzialny za zarządzanie planami treningowymi w aplikacji FitAI.

## Planowana funkcjonalność

- Tworzenie, edytowanie i usuwanie planów treningowych
- Zarządzanie ćwiczeniami w planie
- Zapisywanie logów ukończonych treningów

## Stack

- Python 3.11 + FastAPI
- PostgreSQL (SQLAlchemy + Alembic)

## Port

`8002`
